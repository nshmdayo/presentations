# Vibe Coding with AI

生成AIを使ったVibe Codingについてのスライド資料

## 概要

このプレゼンテーションでは、生成AIを活用したプログラミング手法「Vibe Coding」について説明します。直感的で自然なプログラミングアプローチを通じて、開発効率を大幅に向上させる方法を紹介します。

## 内容

- Vibe Codingとは何か
- 生成AIを使うメリット
- 具体的な活用方法
- 実践的なプロンプトテクニック
- 注意点とベストプラクティス
- 将来展望

## 開発・実行方法

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev

# プレゼンテーションをビルド
npm run build

# PDFエクスポート
npm run export
```
